    音乐包生成器

    简单易懂

    不需要太多繁杂的操作，小白也会用

    快速生成

    自动打包，快速生成

    # 使用方法

    # 第一步

    访问网站蛋白后宫工具箱open in new window

    点击【音乐包生成器】

    # 第二步

    输入【音乐包名称】与【音乐包描述】(可选)

    点击【创建】

    # 第三步

    等待显示【创建成功】后，点击【上传音频】

    上传ogg格式的文件，单个文件不大于3MB，不含中文字符

    点击【开始上传并封装】

    等待系统自动打包

    # 第四步

    等待打包完成后

    点击【查看命令】会显示命令

    点击【下载音乐包并结束制作】会自动下载文件，并在1分钟后删除音乐包，以便后续【重复名称】使用
