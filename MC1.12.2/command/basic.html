                                                                                                                                                                                                                                                # 目标选择器

                                                                                                                                                                                                                                                # 前言

                                                                                                                                                                                                                                                所有标注<玩家>的地方都可以是【目标选择器】

                                                                                                                                                                                                                                                # 介绍

                                                                                                                                                                                                                                                目标选择器，字面意思，可以选择实体/玩家目标

                                                                                                                                                                                                                                                # 语法

                                                                                                                                                                                                                                                <目标类型>[属性名=数据]

                                                                                                                                                                                                                                                # 名词解释

                                                                                                                                                                                                                                                # 目标类型

                                                                                                                                                                                                                                                目标类型 简介
                                                                                                                                                                                                                                                @a 选择全部玩家
                                                                                                                                                                                                                                                @p 选择附近一名玩家
                                                                                                                                                                                                                                                @r 随机选择一名玩家
                                                                                                                                                                                                                                                @e 选择全部实体
                                                                                                                                                                                                                                                @s 选择自己

                                                                                                                                                                                                                                                # 属性名与数据

                                                                                                                                                                                                                                                属性名 数据 举例 简介
                                                                                                                                                                                                                                                type 实体名 @e[type=minecraft:pig] 实体-类型
                                                                                                                                                                                                                                                x,y,z 坐标 @e[x=1,y=1,z=1] 实体-位置判定区域中心点
                                                                                                                                                                                                                                                r 实体所在的格子，取半径的数值 @e[r=2] 位置-判定区域半径
                                                                                                                                                                                                                                                m 游戏模式 @p[m=survival] 玩家-游戏模式
                                                                                                                                                                                                                                                c 实体数量 @e[c=10] 实体-数量
                                                                                                                                                                                                                                                lm,l lm=最小等级，l=最大等级 @p[lm=0,l=0] 实体-经验等级
                                                                                                                                                                                                                                                name 实体名字 @p[name=player_name] 实体-名字
                                                                                                                                                                                                                                                team 队伍名称 @p[team=team_name] 实体-队伍
                                                                                                                                                                                                                                                tag 标签名称 @p[tag=tag_name] 实体-标签
                                                                                                                                                                                                                                                score score_(计分板名称)=数值 | _min=最小值 @p[score_name=0,score_name_min=0] 计分板-数值
                                                                                                                                                                                                                                                rym,ry,rxm,rx 水平 rym=最小值,ry=数值 | 垂直rxm=最小值,rx=数值 @p[rxm=0,rx=0] | @p[rym=0,ry=0] 玩家-视角朝向
                                                                                                                                                                                                                                                dx,dy,dz 坐标 @p[dx=0,dy=0,dz=0] 位置-判定区域延伸

                                                                                                                                                                                                                                                # 命令解释

                                                                                                                                                                                                                                                # 成就/进度

                                                                                                                                                                                                                                                # 语法

                                                                                                                                                                                                                                                /advancement <grant|revoke|test> <玩家> <everything|from|only|through|until> <进度/成就名>

                                                                                                                                                                                                                                                # 名词解释

                                                                                                                                                                                                                                                属性名 说明
                                                                                                                                                                                                                                                grant 给予
                                                                                                                                                                                                                                                revoke 移除
                                                                                                                                                                                                                                                test 检测
                                                                                                                                                                                                                                                everything 全部成就
                                                                                                                                                                                                                                                from 从x成就（母成就）向下的所有成就（子成就）
                                                                                                                                                                                                                                                only 特定的唯一成就
                                                                                                                                                                                                                                                through 从x成就向上向下的所有成就
                                                                                                                                                                                                                                                until 从x成就（子成就）向上的所有成就（母成就）

                                                                                                                                                                                                                                                # 命令举例

                                                                                                                                                                                                                                                让<玩家>获得所有成就
                                                                                                                                                                                                                                                /advancement grant <玩家> everything

                                                                                                                                                                                                                                                让<玩家>获得[探索的时光]和向下延申的所有成就
                                                                                                                                                                                                                                                /advancement grant <玩家> from minecraft:adventure/adventuring_time

                                                                                                                                                                                                                                                让<玩家>移除[探索的时光]单一成就
                                                                                                                                                                                                                                                /advancement revoke <玩家> only minecraft:adventure/adventuring_time

                                                                                                                                                                                                                                                让<玩家>移除[探索的时光]向上向下的所有成就
                                                                                                                                                                                                                                                /advancement revoke <玩家> through minecraft:adventure/adventuring_time

                                                                                                                                                                                                                                                检测<玩家>是否拥有[探索的时光]及向上的所有成就
                                                                                                                                                                                                                                                /advancement test <玩家> until minecraft:adventuer/adventuring_time

                                                                                                                                                                                                                                                # /clear

                                                                                                                                                                                                                                                # 命令解释

                                                                                                                                                                                                                                                # 清空背包

                                                                                                                                                                                                                                                # 语法

                                                                                                                                                                                                                                                /clear <玩家> <物品> <数据> <最大量> <物品NBT>

                                                                                                                                                                                                                                                # 名称解释

                                                                                                                                                                                                                                                <物品NBT>

                                                                                                                                                                                                                                                # 命令举例

                                                                                                                                                                                                                                                假如你背包里有张纸，清除这一张纸的指令就是
                                                                                                                                                                                                                                                /clear <玩家> minecraft:paper 0 1

                                                                                                                                                                                                                                                假如你背包里有张纸，它的名字叫[一张纸]，单独清除这一张纸的指令就是
                                                                                                                                                                                                                                                /clear <玩家> minecraft:paper 0 1 {display:{Name:"一张纸"}}

                                                                                                                                                                                                                                                # 小知识

                                                                                                                                                                                                                                                /clear 不仅可以清除物品，还可以用于检测

                                                                                                                                                                                                                                                如果想判断玩家背包里是否有张纸，指令就是
                                                                                                                                                                                                                                                /clear <玩家> #minecraft:paper 0 0

                                                                                                                                                                                                                                                # /clone

                                                                                                                                                                                                                                                # 命令解释

                                                                                                                                                                                                                                                # 区域复制

                                                                                                                                                                                                                                                # 语法

                                                                                                                                                                                                                                                /clone <x1 y1 z1> <x2 y2 z2> <x y z> <filtered|masked|replace> <force|move|normal>

                                                                                                                                                                                                                                                # 名词解释

                                                                                                                                                                                                                                                属性名 说明
                                                                                                                                                                                                                                                filetered 不复制指定方块
                                                                                                                                                                                                                                                masked 遮罩模式 仅复制非空气方块 保持目的区域原本会被替换为空气的方块不变 可以理解为二者重叠
                                                                                                                                                                                                                                                replace 复制所有方块 会用你选中区域所有方块替换目的区域
                                                                                                                                                                                                                                                force 强制复制，无论你复制的位置会不会与原来的区域重合
                                                                                                                                                                                                                                                move 复制并将源位置替换空气

                                                                                                                                                                                                                                                当分组为false是直接把玩家在一个范围内随机扩散，不分队伍。

                                                                                                                                                                                                                                                # 设置世界时间

                                                                                                                                                                                                                                                /time query day

                                                                                                                                                                                                                                                在第三/最后一个括号中，左边的数字是【水平/横向朝向】，右边的数字是【垂直/纵向朝向】

                                                                                                                                                                                                                                                /tp @a 0 5 0

                                                                                                                                                                                                                                                # 世界边界

                                                                                                                                                                                                                                                # 语法

                                                                                                                                                                                                                                                /worldborder <add|ceter|damage|get|set|warning> <数值> <时间>

                                                                                                                                                                                                                                                属性名 说明
                                                                                                                                                                                                                                                add 边界半径增加

                                                                                                                                                                                                                                                get 读取时间边界的宽度

                                                                                                                                                                                                                                                /worldborder center 1 0
