          # 游戏目录

          在启动器主页面左边，点击左下的【版本选择】进入

          添加或导入中，选择第一项【添加已有文件夹】

          在弹出的窗口里，选择【游戏根目录】（.minecraft）

          # 游戏设置

          在启动器主页面左边，点击右下的【版本选择】进入

          点击【设置】

          名称 描述 作用
          游戏Java Java所在位置 设置Java所在位置，会自动寻找
          版本隔离 版本隔离 把当前游戏目录下多个游戏版本，分开存放
          游戏内存 游戏可使用的内存 设置游戏可使用的内存，根据实际内存大小，微调，越大越好，⚠但不要超出总内存
          文件校验 启用或禁用文件校验 关闭该选项后，启动游戏时会跳过校验文件完整性，直接启动，反之（一般情况下不要动它）。

          # 启动游戏

          在启动器主页面左边，点击【启动游戏】即可
